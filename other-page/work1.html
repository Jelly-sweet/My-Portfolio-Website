制作概要

20ルールタイマー（眼精疲労予防）・ポモドーロタイマー・読書用タイマー・ストレッチ用タイマーの4種類を一つにまとめたアプリを制作しました。 シンプルで見やすいUIを意識し、作業中でも残り時間を直感的に把握できるように設計。通知音やイラストの変化でタイマー終了をわかりやすく伝えています。 デザインやイラストも一から自作し、全体の世界観を統一しました。

担当内容：企画、デザイン、イラスト、コーディング
製作期間：約2週間（2025.08.19-2025.09.05）
使用技術：IbisPaint / Figma / HTML / CSS / JavaScript

  • コンセプト

    波に身を任せて生きるクラゲのように、無理せず自然体で続けられる時間を大切にしています。 ポモドーロ・読書・ストレッチなど、心をゆるめながら集中できるツールです。

  • 目的

    自宅に時計がなく、スマホのタイマーを使うとついSNSなどを開いてしまうことが多かったため、スマホを見ずにPC上で作業時間を管理できるタイマーアプリを制作しました。

①20ルールタイマー

「20分の作業ごとに20秒間、20フィート先を見る」という一連の動作が、長時間の画面作業による目の疲れを防ぐのに効果的だといわれていることから、このタイマーを作成しました。 作業中のイラストはPC作業の様子を、20秒間の休憩中のイラストは外を眺めている様子を描いています。 また、リセットボタン・ポーズボタン・閉じるボタンを押すまで、この動作が繰り返されるようになっています。

Start

Finish

②ポモドーロタイマー

25分の集中作業と5分間の休憩を繰り返す『時間管理術（タイムマネジメント法）』を取り入れたタイマーです。 ドーナツが円状に回転し、25分経つと口元に到達し食べれるようになる演出にしています。 また、リセットボタン・ポーズボタン・閉じるボタンを押すまで、この動作が繰り返されるようになっています。

Start

Finish

③読書タイマー

普段から1日10分の読書時間を確保するよう心がけていることから、読書用のタイマーを制作しました。 リラックスして読書している様子をイラストにしています。 1回限りのカウントダウン機能となっており、再度使用する場合はスタートボタンを押すことで再びタイマーが起動します。

Finish

④ストレッチタイマー

就寝前のストレッチ用として作成しました。 ストレッチしているイラストから始まり、終了時は寝ているイラストに切り替わるようにしています。 1回限りのカウントダウン機能となっており、再度使用する場合はスタートボタンを押すことで再びタイマーが起動します。
