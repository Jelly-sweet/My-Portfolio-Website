制作概要

ポートフォリオ作品として、デザインから実装まで一貫制作しました。
ダークトーンを基調に、静かなアニメーションで視覚的奥行きと世界観の統一感を演しています。

制作範囲：企画、デザイン、コーディング
製作期間：約4週間（2025.09.20 - 2025.10.15）
使用技術：Figma / HTML / CSS / JavaScript

  • コンセプト

    全体をダークトーンで構成し、静けさと落ち着きを感じられるデザインにしました。 その中にアクセントとして淡い光や滑らかなアニメーションを加えることで、動きや温かみを表現しています。 「派手さよりも印象に残る静かなサイト」 をテーマに、シンプルながらも視線が自然に誘導される構成を意識しました。

  • 目的

    web業界への転職を行うためにポートフォリオが必要。
    デザインやコーディングなど、今の自分ができることを最大限アピールしたい。 最終ゴールは内定の獲得。

  • ターゲット

    採用担当者、デザイナー、ディレクター、コーダーの方々。
