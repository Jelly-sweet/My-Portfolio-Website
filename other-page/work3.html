制作概要

HTML / CSS / JavaScript の基礎を学習したのち、デザインからマークアップまでを初めて一貫して行い制作した架空のWebサイトです。

制作範囲：企画、デザイン、コーディング
製作期間：4日（2025.08.04 - 2025.08.07）
使用技術：Figma / HTML / CSS / JavaScript

  • コンセプト

    Calix（杯）が象徴する“もてなし”と“美しさ”をコンセプトに、 ゆったりとした時間を楽しむ大人のラウンジ空間をデザインしました。 深い黒に金のアクセントを添え、上質で落ち着いた雰囲気を演出しています。

  • 目的

    HTML/CSS/JavaScript の基礎学習と、7月に行った13本の模写経験を踏まえ、当時の自分の実力を確認することを目的に制作しました。 デザインからマークアップまで一貫して制作することで、理解度と応用力をチェックしています。

  • ターゲット

    ラグジュアリーな雰囲気を好む大人（20代後半〜40代前後）、落ち着いた空間でゆったりした時間を楽しみたい人向け
